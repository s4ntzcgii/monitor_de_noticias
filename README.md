# 📰 Monitor de Notícias em Tempo Real

## 📌 Objetivo da Atividade

Este projeto foi desenvolvido em Node.js com o objetivo de realizar Web Scraping em um portal de notícias, coletando manchetes automaticamente e armazenando os dados em um banco SQLite.

O sistema simula um monitoramento de tendências de publicação utilizado por agências de comunicação.

---

# 🛠️ Tecnologias Utilizadas

- Node.js
- Axios
- Cheerio
- SQLite3
